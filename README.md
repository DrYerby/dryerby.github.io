# Johnathan Yerby, Ph.D

This repository contains a static Astro website for the professional profile of Johnathan Yerby, Ph.D.

## Development

Use the following commands to work locally:

```bash
npm.cmd install
npm.cmd run dev
```

## Build

To create a production build:

```bash
npm.cmd run build
```

The site is configured for GitHub Pages deployment via .github/workflows/deploy.yml.
